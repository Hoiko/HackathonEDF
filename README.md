# Medecin du Monde
React 16 / redux with Syphony


## Install
Front only

```sh
cd frontend && npm install
```

Api only

```sh
cd api && php bin/console composer install
```

Run both of them

```sh
make install
```

## Run
Front only

```sh
cd frontend && npm start
```

Api only

```sh
cd api && php bin/console server:start
```

Run both of them

```sh
make run
```
